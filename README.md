# Moja Vercel vaja

To je enostavna statična spletna aplikacija, izdelana za vajo pri predmetu UOS.

## Funkcionalnosti
- naslov strani in kratek opis,
- gumb za interakcijo,
- števec klikov,
- prikaz trenutnega časa,
- preprost obrazec brez backenda.

## Originalna verzija
Originalna verzija projekta je vsebovala naslov, kratek opis in en gumb, ki je ob kliku spremenil besedilo na strani.

## Nadgrajena verzija
Nova verzija poleg osnovne spremembe besedila vključuje še števec klikov, digitalno uro in preprost obrazec z JavaScript preverjanjem vnosa brez backenda.

## Uporabljene tehnologije
- HTML
- CSS
- JavaScript

## Namen projekta
Namen projekta je prikazati osnovni postopek priprave spletne aplikacije, objave na GitHub in deploya na Vercel.

## Deploy
Projekt se objavi prek GitHub repozitorija in poveže z Vercelom za produkcijski deploy.
